这是一个自用的记事本软件, 文本编辑是基于pyqt5的QTextEdit组件, 基本功能copy自 https://github.com/goldsborough/Writer-Tutorial, 运行环境因为本人在使用 anaconda, 这个包里已经自带了 pyqt5, 所以如果对环境有别的喜好请慎用.
学习pyqt5的使用过程中...

基本功能:
1. 工作目录的简单管理, 左边是目录树
2. 从剪贴板的贴上图片, 并将图片保存到文档目录中
3. 插入的图片文件自动copy到文档目录下
4. 实现文本文档的加密功能
5. 文档目录与git同步(菜单手动)
